# 食材管理アプリ

食材の賞味期限をガントチャート形式で視覚的に管理するWebアプリケーションです。

## 特徴

- **ガントチャート表示**: 食材と賞味期限を時間軸上で一目で把握
- **期限アラート**: 賞味期限までの日数に応じて色分け表示
  - 🟢 緑: 8日以上（新鮮）
  - 🟡 黄色: 4-7日（注意）
  - 🟠 オレンジ: 1-3日（要注意）
  - 🔴 赤: 期限切れ
- **CRUD機能**: 食材の追加・編集・削除が簡単
- **検索・フィルター**: 食材名で検索、カテゴリーやステータスでフィルター
- **データ永続化**: ブラウザのlocalStorageに自動保存
- **レスポンシブデザイン**: デスクトップ・タブレット・モバイルに対応

## 使い方

### インストール

1. このディレクトリをブラウザで開きます

```bash
cd /Users/akibanomori/Sites/food-ingredient-manager/
open index.html
```

または、ダブルクリックで `index.html` を開きます。

### 基本操作

#### 食材の追加

1. 「+ 食材を追加」ボタンをクリック
2. フォームに食材情報を入力
   - 食材名（必須）
   - カテゴリー（必須）
   - 購入日（任意）
   - 賞味期限（必須）
   - 数量（任意）
   - 保管場所（任意）
   - メモ（任意）
3. 「保存」ボタンをクリック

#### 食材の編集

1. ガントチャート上の食材の「編集」ボタンをクリック
2. フォームで情報を修正
3. 「保存」ボタンをクリック

#### 食材の削除

1. ガントチャート上の食材の「削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック

#### 検索・フィルター

- **テキスト検索**: 検索ボックスに食材名を入力
- **カテゴリーフィルター**: ドロップダウンからカテゴリーを選択
- **ステータスフィルター**: ドロップダウンからステータスを選択
- **クリア**: 「クリア」ボタンですべてのフィルターを解除

## 機能詳細

### ガントチャート

- **表示期間**: 今日から3ヶ月（90日）先まで
- **タイムラインバー**: 購入日から賞味期限までの期間を視覚化
- **色分け**: 賞味期限までの日数に応じて自動的に色が変わります
- **今日インジケーター**: 現在の日付を青い線で表示

### データ管理

- **自動保存**: 食材の追加・編集・削除時に自動的にlocalStorageへ保存
- **容量**: 約10,000件以上の食材を保存可能（5-10MB制限内）
- **バックアップ**: コンソールコマンドでJSON形式でエクスポート可能

### バリデーション

- **必須項目**: 食材名、カテゴリー、賞味期限
- **日付チェック**: 購入日が賞味期限より後でないか確認
- **文字数制限**: 食材名は100文字以内

### アクセシビリティ

- セマンティックHTML
- ARIAラベル
- キーボード操作対応（Tab、Enter、ESC）
- フォーカス管理

## 開発

### デバッグ

### ローカルストレージの確認

開発者ツール > Application > Local Storage から `foodIngredients` キーを確認できます。

## トラブルシューティング

### データが保存されない

- ブラウザのlocalStorageが有効か確認
- プライベートブラウジングモードでは保存できません
- ストレージ容量を確認（`logStats()` コマンド）

### 表示が崩れる

- ブラウザのキャッシュをクリア
- 最新版のブラウザを使用
- JavaScriptが有効か確認

### 食材が表示されない

- コンソールエラーを確認
- `loadIngredients()` コマンドでデータを確認
- ブラウザをリロード
